## 🌸 Flower Image Classification using CNN

A deep learning project built using Python and Jupyter Notebook that classifies flower images into different species using Convolutional Neural Networks (CNNs). The project demonstrates image preprocessing, data augmentation, CNN architecture design, model training, evaluation, and prediction using a real-world flower image dataset.

## 🚀 Features
* **Flower Species Classification** – Classify flower images into multiple categories.
* **Convolutional Neural Network (CNN)** – Build and train a deep learning image classifier.
* **Image Preprocessing Pipeline** – Resize, normalize, and prepare images for training.
* **Data Augmentation** – Improve model generalization using image transformations.
* **Multi-Class Image Classification** – Predict flower species across multiple classes.
* **Model Training & Evaluation** – Train CNN models and evaluate prediction performance.
* **Prediction Visualization** – Display predicted classes alongside sample images.
* **Performance Metrics Analysis** – Analyze accuracy and validation performance.
* **Real-World Flower Dataset** – Practice computer vision using thousands of labeled flower images.
* **End-to-End Deep Learning Workflow** – Covers preprocessing, training, evaluation, and inference.

## 🛠️ How to Run
1. **Clone the repository**:
```
git clone https://github.com/Agent-A345/Flower-Image-Classification-CNN.git
```
2. **Navigate to the project folder**:
```
cd Flower-Image-Classification-CNN
```
3. **Install required libraries**
```
pip install tensorflow keras numpy pandas matplotlib opencv-python scikit-learn jupyter
```
4. **Run the notebook**
```
jupyter notebook
```
5. **Open**
```
image_classification_cnn.ipynb
```
